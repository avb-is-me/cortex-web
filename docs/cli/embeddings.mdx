# `cortex embeddings`

This command creates the embedding vector representing the input text.

:::info
This command is compatible with all OpenAI and OpenAI-compatible endpoints.
:::

## Usage

:::note
Ensure you have loaded the model into the Cortex server using [`cortex models start`](/docs/cli/models/start) before calling the embeddings command.
:::

```bash 
# With a model started
cortex embeddings [flags] [message]
# Without any started models
cortex embeddings [flags] [model_id] [message]
```

## Options


| Option                                    | Description                                                                                                             | Required | Default value        |
|-------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|----------|----------------------|
| `-m, --model <model_id>`                  | Specify the models to generate the embeddings.                | No       | Active models (if any) |
| `-i, --input <input>`                     | Input text to embed, encoded as a string or array of tokens. For multiple inputs, pass an array of strings or token arrays. | Yes      |    -                  |
| `-e, --encoding_format <encoding_format>` | Encoding format for the embeddings. Supported formats are float and int.                              | No       | `float`                |
| `-d, --dimensions <dimensions>`           | Specify the number of dimensions for the resulting output embeddings. Supported only in some models.                     | No       |     -                 |
| `-h, --help`                              | Display help information for the command.                                                                                 | No       |     -                 |
