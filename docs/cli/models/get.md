# `cortex models get`

This command returns a model detail defined by a `MODEL_ID`.

:::info
This command is compatible with all OpenAI and OpenAI-compatible endpoints.
:::

## Usage

```bash
cortex models get MODEL_ID
```

## Options

```
  -h, --help              Display help for command
```
