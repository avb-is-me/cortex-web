# `cortex models stop`

This command stops a model defined by a `MODEL_ID`.

:::info
This command is compatible with all OpenAI and OpenAI-compatible endpoints.
:::

## Usage

```bash
cortex models stop MODEL_ID
```

## Options

```
  -h, --help              display help for command
```
