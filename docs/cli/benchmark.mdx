# `cortex benchmark`

This command benchmarking your hardware to analyze the selected model performance on your system.

:::info
This command is compatible with all OpenAI and OpenAI-compatible endpoints.
:::

## Usage

```bash
cortex benchmark
```
:::info
To benchmark a specific model, edit `model.yaml` in `~/cortex/benchmark`. A feature to select models for benchmarking will be added soon.
:::


## Options

| Option                           | Description                                                                 | Required |
|----------------------------------|-----------------------------------------------------------------------------|----------|
| `-n, --num_rounds <num_rounds>`  | Number of rounds to run the benchmark. Default value is `10` rounds.                                        | No      |
| `-c, --concurrency <concurrency>`| Number of concurrent requests to run the benchmark. Default is `false`.                           | No      |
| `-o, --output <output>`          | Output format for the benchmark results. json or table. Default is `json`.                       | No      |
| `-h, --help`                     | Display help information for the command.                                                     | No       |
